## React

For general installation and configuration look at the [react-components](https://www.npmjs.com/package/@db-ui/react-components) package.

### Use component

```tsx App.tsx
// App.tsx
import { DBLink } from "@db-ui/react-components";

const App = () => (
	<DBLink href="#" variant="primary">
		Link
	</DBLink>
);

export default App;
```
